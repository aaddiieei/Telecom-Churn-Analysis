# Telecom Churn Analysis 📊

This project analyzes telecom customer data to predict churn and identify key factors that lead to customer attrition.

## Project Structure
* `TCA.ipynb`: The main Python notebook containing Data Cleaning, EDA, and Modeling.
* `Customer Churn.csv`: The dataset used for analysis.
* `Telecom Customer Churn Analysis.docx`: Detailed project report.

## Tools Used
* **Python** (Pandas, Matplotlib, Seaborn)
* **Google Colab**
* **Machine Learning** (Logistic Regression/Random Forest)

##  Key Insights
* Identified top reasons for customer churn.
* Provided recommendations for customer retention.
